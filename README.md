# Mysh
## 간단한 쉘 개발
> C 언어 기반의 간단한 쉘 프로그램으로, 내부 명령어와 외부 명령어를 모두 처리할 수 있으며, 파이프(`|`), 리다이렉션(`>`, `>>`), 백그라운드 실행(`&`) 기능을 지원합니다.

---

## 주요 기능
- 외부 명령어 : `ls`, `cat`, `rm`, `ps`, `whoam`, `date` 등
- 내부 명령어 : `help`, `exit`, `quit`
- 리다이렉션(`>`, `>>`) : 표준 출력을 파일로 덮어쓰기 or 추가하기
- 파이프(`|`) : 한 프로세스의 표준 출력을 다른 프로세스의 표준 입력으로 전달
- 백그라운드 실행(`&`) : 명령어를 백그라운드에서 실행

---

## 실행 (Ubuntu)
```bash
gcc -o mysh mysh.c

./mysh
```
